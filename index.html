本站的构建

文本编辑是 Emacs 中的 Org-mode 文件，具体详见采用 Org-publish 发布博客。

Vim 编辑器被称为编辑器之神，而 Emacs 被称为神的编辑器。

Emacs 基于 Emacs lisp 可以实现很多的功能，是被伪装的操作系统。在 Emacs 里可以进行笔记管理、事务管理、听音乐、收邮件、聊天等，最主要的还是文本编辑。
